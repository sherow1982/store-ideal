# Store Ideal - متجر إلكتروني فاخر

## الملفات المضمنة:
- style.css: ملف الثيم الموحد (ذهبي فاتح مبهر)
- index.html: الصفحة الرئيسية
- about.html: صفحة عن المتجر
- contact.html: صفحة الاتصال
- privacy.html: سياسة الخصوصية
- terms.html: شروط الخدمة
- products/product.html: صفحة منتج العينة

## التعليمات:
1. ضع جميع الملفات في مجلد واحد
2. كل ملف HTML يشير إلى style.css عبر رابط موحد
3. أي تعديل على style.css سيظهر في كل الصفحات تلقائياً

## المتطلبات:
- متصفح حديث
- لا توجد متطلبات أخرى!

## النسخ للاستخدام:
1. انسخ جميع الملفات للمجلد الخاص بك
2. افتح index.html في المتصفح
3. عدّل style.css حسب احتياجاتك

## ملاحظة:
كل صفحة تستخدم نفس style.css للتصميم الموحد.
لتغيير الألوان أو الأسلوب، عدّل فقط ملف style.css!
